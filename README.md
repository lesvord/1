# Stalker Telegram Chat Game

Прототип чат-игры в стиле сталкера для Telegram. Управление — кнопками меню и инлайн-кнопками.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Что есть сейчас

- Главное меню с кнопками.
- Инлайн-кнопки для интерактивных действий.
- Базовые заглушки профиля, локаций, рейдов.
